# Interaction crop gene editing regulations tracker

Gene editing regulations are changing globally and are often difficult to keep track of. While there is a rich publication on these changes, publications the rate of change means that these resources are quickly out of date.

Our website cropgeregulation.com helps users quickly see and share maps tracking regulatory change for gene editing. The first iteration of this site used static maps, which are ideal for publications, slides and reports, but an interactive map could provide more information to uses.

This is the prototype test to create an interactive map for cropgeregulation.com, allowing users to quickly browse global regulatory status for gene edited crops.

## Planned features
